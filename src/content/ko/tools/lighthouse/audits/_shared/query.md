아래 검사에서, Lighthouse는 `1 element fails this test` 같은 걸 보여줍니다.
숫자는 실패한 요소의 갯수에 따라 다릅니다. 목록을 확장하려면 이 레이블을 클릭하세요.
Chrome DevTools 콘솔에서 [`$()`][qs], [`$$()`][qsa], and [`$x()`][xp] 함수를 실행하여
DOM에서 각 요소를 찾을 수 있습니다.
