# Web Fundamentals {: .page-title }

## 웹 푸시 알람은 사용자를 유치하기 위한 새로운 방법 {: .hide-from-toc  }

![푸시 알람 예제](/web/images/common/push-notification-16x9.png){: .attempt-right }

웹 푸시 알람은 적절한 타이밍에 문맥과 연관 있는 푸시 알람을 보내줌으로써 사용자를 다시 사이트로 유입시킵니다.
심지어 브라우저가 닫혀 있을 때도 동작하죠!

웹 푸시 알람은 사용자가 시기적절하게 좋아하는 사이트의 업데이트를 할 수 있게 하고,
사용자가 관심 있을만한 개인화된 컨텐츠로 사이트 재진입을 효과적으로 유도할 수 있습니다.

푸시 API 와 알람 API 로 당신의 사용자와 재 교류할 수 있는 새로운 가능성을 엽니다.
