Introduction aux Applications Web Progressives

Dans ce cours, vous allez commencer à travailler sur votre toute première Application Web Progressive - une application Web qui peut profiter de nombreuses fonctionnalités d'applications natives. Vous aurez également plus d'expérience dans la création d'une application Web fonctionnant hors ligne à l'aide de Service Workers. Enfin, vous rendrez votre application installable sur l'écran d'accueil de l'utilisateur avec le fichier Manifest.
