Suivez ces étapes dans les cas ci-dessous :

  • Vous avez été averti par Google que votre site a été piraté.
  • Un message d'avertissement s'affiche dans les résultats de recherche et vous informe que votre site a été piraté ou infecté.
  • Vous ignorez si votre site a réellement été piraté.

Dans ce dernier cas, ou si vous estimez que votre site a été signalé de façon incorrecte, enregistrez-le dans la Search Console. Dans la section Actions manuelles ou Problèmes de sécurité de la Search Console, recherchez des exemples d'URL pour lesquelles nous avons constaté que votre site a été piraté.

Si vous ne trouvez aucun contenu piraté dans les URL indiquées dans la Search Console, il se peut que le contenu piraté utilise une technique de dissimulation connue sous le nom de cloaking. Cette technique rend le nettoyage du site plus difficile, puisqu'elle affiche un contenu différent à différents types d'utilisateurs. Par exemple, lorsque vous accédez à une page de votre site comme www.example.com/cheap-drugs, une page vide peut s'afficher. Cela peut vous amener à penser qu'il n'y a pas de contenu piraté sur votre site. Cependant, si un moteur de recherche comme Google accède à la page www.example.com/cheap-drugs, il affichera les mots et les liens qui contiennent du spam.

Pour vérifier si la technique de cloaking a été employée, utilisez l'outil de dépannage pour les sites piratés. Cet utilitaire met à votre disposition différents outils, tels que l'opérateur de recherche site: et Explorer comme Google, pour vous aider à découvrir les contenus piratés par cloaking.

Après avoir vérifié vos pages, si vous continuez à penser que votre site a été mal signalé, veuillez l'indiquer dans les Forums d'aide aux webmasters. Sinon, suivez notre guide d'aide à la récupération des sites piratés pour les webmasters afin de nettoyer votre site.
