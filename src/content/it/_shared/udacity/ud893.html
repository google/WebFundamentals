Web Design Responsive

In questo corso imparerai i fondamenti del design web responsive con Pete LePage di Google! Creerai la tua pagina web responsive che funziona bene su qualsiasi dispositivo - telefono, tablet, desktop o qualsiasi cosa in mezzo.

Inizierai ad esplorare ciò che rende un sito responsive e come funzionano alcuni noti design patterns responsive per diversi dispositivi. Da questi imparerai a creare il tuo responsive layout utilizzando il tag viewport e le query CSS media. Mentre procedi, sperimenterai con i grandi e piccoli breakpoint, ed ottimizzerai il testo per la lettura.
