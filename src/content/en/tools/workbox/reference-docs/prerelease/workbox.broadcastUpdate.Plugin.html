workbox.broadcastUpdate.Plugin

class   static
Version     v4.0.0-rc.4

This plugin will automatically broadcast a message whenever a cached response is updated.

Constructor

Plugin

new Plugin(options)

Construct a BroadcastCacheUpdate instance with the passed options and calls its notifyIfUpdated() method whenever the plugin's cacheDidUpdate callback is invoked.

Parameter

options

Object

Values in options have the following properties:

Parameter

headersToCheck

Optional

Array of string

A list of headers that will be used to determine whether the responses differ.

channelName

Optional

string

The name that will be used . when creating the BroadcastChannel, which defaults to 'workbox' (the channel name used by the workbox-window package).

deferNoticationTimeout

Optional

string

The amount of time to wait for a ready message from the window on navigation requests before sending the update.
