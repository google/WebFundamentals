workbox-sw

A high-level library that makes it easier to precache assets and configure routes with caching strategies in a service worker.

Classes

Router

Adds a friendly API on top of the router from the workbox-routing module.

Strategies

This is a simple class used to namespace the supported caching strategies in workbox-sw.

You would never access this class directly but instead use with workboxSW.strategies.<Strategy Name>.

WorkboxSW

This class uses the Workbox libraries to create a clean and easy API for common caching and serving needs.
