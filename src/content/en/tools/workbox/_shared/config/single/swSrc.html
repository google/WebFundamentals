swSrc

Required String

The path to the source service worker file that can contain your own customized code, in addition to containing a match for injectionPointRegexp.

In Node

The file should include a call to a workbox.precaching method that makes use of the injected precache manifest.

In Webpack

Your sw file should ycan reference the self.__precacheManifest variable to obtain a list of ManifestEntrys obtained as part of the compilation: workbox.precaching.precacheAndRoute(self.__precacheManifest)

Example: 

swDest: path.join('src', 'sw.js')
