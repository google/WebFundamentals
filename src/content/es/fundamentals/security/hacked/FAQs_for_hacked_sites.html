En este artículo, hemos recopilado las respuestas a las preguntas más frecuentes sobre piratería informática que recibimos en Google.

¿Por qué han pirateado mi sitio web?

Los hackers pueden piratear un sitio web por alguno de estos motivos:

  • Obtener beneficios económicos mediante el spam web: sitúan enlaces de spam en tu sitio web o redirigen el tráfico a sitios web externos.
  • Robar información confidencial: copian archivos que contienen datos de clientes, como números de tarjetas de crédito, información de identificación personal o credenciales de inicio de sesión.
  • Comunicar un mensaje político o social: desfiguran un sitio web como forma de hacktivismo para destacar un punto de vista y provocar el cambio.
  • Insertar software malicioso: insertan código malicioso mediante secuencias de comandos o iFrames que extraen contenido de otro sitio web y que intenta atacar a cualquier ordenador que vea la página.
  • Buscar emociones fuertes o causar vandalismo: no existe ningún motivo especial; solo quieren allanar tu sitio web y causar daños.

¿Cómo puedo saber si me han pirateado?

Para saber si te han pirateado el sitio web, busca los siguientes indicios habituales: picos de tráfico inusuales (sobre todo de términos de búsqueda que no estén relacionados con la temática de tu sitio web), visitantes que informen de software malicioso, cuentas creadas recientemente que cuenten con privilegios de administrador o páginas nuevas sospechosas que se hayan añadido al sitio web.

¿Cómo me han pirateado?

Es posible que te hayan pirateado el sitio web por alguna de estas causas: vulnerabilidades en el software, contraseñas filtradas o adivinadas, páginas de administrador que no requieran autenticación, consultas de base de datos sin limpiar o puertos abiertos innecesarios. También es posible que hayan utilizado técnicas de ingeniería social (como el phishing o la suplantación de identidad de una autoridad de confianza) para aprovecharse de las debilidades humanas. Es fundamental eliminar estas vulnerabilidades o mitigarlas.

¿A quién puedo consultar si tengo más preguntas?

Los foros de ayuda para webmasters tienen un grupo activo de Googlers y colaboradores técnicos que pueden ayudarte proporcionándote su opinión. Además, la mayoría de los proveedores principales de sistemas de gestión de contenido (CMS) disponen de documentación detallada sobre cómo resolver casos de piratería. También puedes pedir ayuda a un profesional de seguridad de confianza.
