Este paso se aplica a los sitios web que se han pirateado para que alojen spam. A menudo se muestra la advertencia "Este sitio puede haber sido comprometido" en los resultados de búsqueda. Es uno de los pasos más largos del proceso de recuperación. En este paso compilaremos una lista con los archivos dañados del sitio web. Utilizaremos esta lista en un paso posterior: Limpiar el sitio web y hacer tareas de mantenimiento.

Si tu sitio web se ha visto afectado por software malicioso y se ha marcado con el mensaje de advertencia "Este sitio puede dañar tu ordenador" en los resultados de búsqueda, consulta el artículo Evaluar los daños provocados por software malicioso (sitio web pirateado con software malicioso).

Determinadas campañas de piratería afectan a miles de sitios web de manera similar, por lo que se pueden limpiar siguiendo una serie de pasos específicos. Si alguna de estas campañas de piratería describe el ataque que ha sufrido tu sitio web, haz clic en ella para leer la guía.

  • Corregir el ataque con palabras clave en japonés
  • Corregir el ataque de texto autogenerado
  • Corregir el ataque con palabras clave y enlaces encubiertos
